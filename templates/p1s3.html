Pseudocode

Pseudocode for finding the largest number in a list of the numbers: 

      INPUT: A list of numbers (e.g., [5, 8, 3, 10, 12, 2])

      SET largest = first number in the list
      
      FOR each number in the list:
          IF number > largest:
              SET largest = number
      
      OUTPUT largest

The Pseudocode helps us understand the logic behind the program before writing the actual code.
